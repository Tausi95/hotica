## How the X funnel works

Every X post = **1 main tweet + 2 replies posted immediately after.**
Image stops the scroll → thread builds desire → final reply has the creator link.

| Time | Format | Goal |
|------|--------|------|
| 8–9am | Hot take / earnings stat (no image) | Shares, reach |
| 1pm | Creator thread — image + 2 replies | Profile clicks, subscriptions |
| 7–8pm | Video clip OR second creator thread | Evening conversions |

Full scripts for all 8 sequences are in `x-funnel-tweets.md`.

## Instagram Notes

- **Format:** Portrait 4:5 (1080x1350px). Crop landscape images before uploading.
- **No links in captions** — always "link in bio". Update bio link daily to that day's creator.
- **Reels:** Use `Short Clips/` mp4s trimmed to 15–30 seconds. Add text overlay with hook.
- **Stories:** Screenshot the best-performing X post each day and repost as a Story.
- **Hashtags (every post):**
  `#hotica #adultcreator #exclusivecontent #creatoreconomy #UKcreators`
- Use the lifestyle copy from the CSV (the 12pm Instagram column) as captions — it's written for Instagram tone, not X.
